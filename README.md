# Linux-Notes

Comprehensive personal notes for learning and referencing Linux fundamentals, commands, administration, and system internals.

**Purpose:** Quick reference and study notes organized by chapter for day-to-day use and interview/study prep.

**How to use:**
- Read chapters in order for a progressive learning path.
- Use the chapter files for examples, command snippets, and hands-on exercises.
- Suggest edits or raise issues via pull requests if you want to contribute.
