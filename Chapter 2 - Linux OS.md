Linux - OS

Kernel is the heart of an Operating System, contains all the necessary files and folders required to run any OS

Linux Kernel is kernel for Linux

Shell - Terminal or platform used to communicate with kernel using shell commands through Shell

Bootloader - GRUB, is the software on Linux that runs and manages the booting process.
GRUB - GRand Unified Bootloader
